# Physical AI & Humanoid Robotics Textbook

This comprehensive textbook teaches students how to build AI-driven humanoid robots using ROS 2, Gazebo, Unity, and NVIDIA Isaac, culminating in a full Vision-Language-Action humanoid capstone project.

## About this Textbook

This textbook is designed for students with backgrounds in Computer Science, Mechatronics, AI/ML, Electrical/Mechanical Engineering. Each chapter includes diagrams, code examples, hands-on labs, and assessments to provide a project-based learning experience.

### Prerequisites

- Python programming experience
- Basic linear algebra and calculus
- Introductory machine learning experience
- Ubuntu 22.04 LTS (for simulation environments)
- ROS 2 Humble Hawksbill installed
